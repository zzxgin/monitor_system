# flask_monitor

#### 介绍
基于Flask开发的轻量级系统监控平台，实现多服务器采集资源监控、关联多用户告警通知和用户登录认证权限管理等功能。
只有管理员可以登录，普通用户的功能是关联服务器告警
● 用户与认证
  ○ 管理员登录（JWT）
  ○ 用户管理：新增、查询、编辑、删除
  ○ 当前用户信息查询
  ○ 重置用户密码
● 服务器管理
  ○ 服务器CRUD
  ○ 用户-服务器多对多关联维护
  ○ 查询指定用户的服务器列表
● 监控数据
  ○ 被监控服务器Agent上报监控数据（API授权认证）
  ○ 查询所有服务器的最新监控数据
  ○ 监控统计：每台服务器的最新CPU/内存/磁盘与阈值
● 告警与通知
  ○ 阈值告警（CPU/内存/磁盘），邮件通知关联用户
  ○ 告警阈值由配置控制
● 安全与跨域
  ○ JWT 鉴权
  ○ API Key 鉴权（Agent上报）
  ○ CORS 放行 /api/*
● 维护与脚本
  ○ 定期清理历史监控数据（cleanup_data.py）
